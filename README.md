# Миний Туслах — Хувийн туслах апп (PWA)

Өдөр тутмын **даалгавар**, **календар**, **зуршил**, **сэрүүлэг/сануулга**-г нэг дор хөтлөх утасны апп. Офлайн ажиллана, утсанд суулгаж болно, бүх мэдээлэл таны төхөөрөмж дээр (localStorage) хадгалагдана — сервер, бүртгэл шаардлагагүй.

## Үндсэн боломжууд
- **🔐 Нэвтрэлт ба эрхийн тохиргоо** — **Gmail хаягаар бүртгүүлж, и-мэйл рүү ирэх кодоор баталгаажуулна** (EmailJS). Нэвтрэх нэр + нууц үгээр нэвтэрнэ. **Нэвтрэх нэр сэргээх** ба **нууц үг сэргээх** — Gmail руу 6 оронтой код илгээж сэргээнэ. Хэрэглэгч бүр **өөрийн тусдаа өгөгдөлтэй**. Админ хэрэглэгчид нэмж, төрөл болон хандах эрхийг (Даалгавар/Календар/Зуршил/Сэрүүлэг/Санхүү) тус бүрчлэн тохируулна. Нууц үг SHA-256 + salt-аар хадгалагдана.
- **🏠 Өнөөдөр** — өнөөдрийн товч мэдээлэл, **цаг агаар**, даалгавар, зуршил, сэрүүлэг.
- **🌦️ Цаг агаар** — байршлаар тань одоогийн температур, нөхцөл, 4 хоногийн урьдчилсан мэдээ (Open-Meteo, түлхүүр шаардлагагүй, үнэгүй). Сүүлийн мэдээ офлайнд хадгалагдана.
- **⚡ Хурдан нэмэх** — "Өнөөдөр" болон "Даалгавар" дэлгэц дээрх мөрөнд бичээд Enter дармагц шууд бүртгэгдэнэ.
- **✓ Даалгавар** — 2 горим: **Өдрөөр** (өдөр тус бүрээр) ба **Бүх жагсаалт** (deadline-аар бүлэглэсэн: хугацаа хэтэрсэн/өнөөдөр/удахгүй/дараа/хугацаагүй). Ангилал (Ажил/Хувийн/Санхүү/Эрүүл мэнд/Суралцах/Бусад), чухал байдал, тэмдэглэл, **deadline тогтоох ба сунгах** (+1 өдөр / +3 / +1 долоо хоног / огноо сонгох). Хийгдсэн/хийгдээгүйгээр шүүх.
- **💰 Санхүү** — бэлэн мөнгө/хадгаламж, зээл, авлага (надад өгөх), өр төлбөр (би өгөх), ирээдүйн орлогыг ангилж бүртгэнэ. **Цэвэр үлдэгдэл (хөрөнгө − өр)**, нийт хөрөнгө, нийт өрийг автоматаар тооцож харуулна.
- **📅 Календар** — сарын харагдац, өдөр сонгож даалгавар/сэрүүлгийг харах, нэмэх.
- **🔥 Зуршил + статистик** — өдөр бүр тэмдэглэх, 🔥 цуваа, 7 хоногийн график. 📊 товчоор: одоогийн/дээд цуваа, нийт тоо, сарын %, **сарын heatmap**.
- **🔔 Сэрүүлэг** — нэг удаа / өдөр бүр / 7 хоног бүр давтах. Цаг болоход мэдэгдэл + дуу + чичиргээ.
- **💾 Нөөцлөх / Сэргээх** — Профайл → *Өгөгдөл* хэсгээс өөрийн бүх өгөгдлийг JSON файл болгон татаж нөөцлөх, дараа нь (өөр төхөөрөмж дээр ч) сэргээх. Сервергүйгээр төхөөрөмж хооронд өгөгдөл зөөх хамгийн энгийн арга.

## Хэрхэн ажиллуулах вэ
Энэ бол энгийн вэб апп тул статик сервер дээр ажиллана.

**Хувилбар 1 — Python (компьютер дээр турших):**
```powershell
cd "C:\Users\user\Desktop\Supporting app"
python -m http.server 5544
```
Дараа нь браузераар `http://localhost:5544` руу орно.

**Хувилбар 2 — Утсандаа суулгах (PWA):**
1. Аппыг интернэтэд байршуулна (жишээ: GitHub Pages, Netlify, Vercel — үнэгүй). Эсвэл компьютер, утас нэг Wi-Fi-д байгаа бол дээрх серверийн хаягийг (`http://<компьютерийн-IP>:5544`) утаснаасаа нээнэ.
2. Утасны браузер (Chrome/Safari) дээр нээгээд:
   - **Android (Chrome):** цэс → *Add to Home screen / Үндсэн дэлгэцэд нэмэх*.
   - **iPhone (Safari):** Share товч → *Add to Home Screen*.
3. Дараа нь үндсэн дэлгэцээс жинхэнэ апп шиг нээгдэнэ.

> 🔔 **Сэрүүлэг анхаар:** Браузерийн мэдэгдэл (notification) зөвшөөрөл хэрэгтэй. Сэрүүлэг нь апп нээлттэй эсвэл арын дэлгэцэд байх үед хамгийн найдвартай ажиллана. Жинхэнэ "дэлгэц унтарсан үед дуугардаг" сэрүүлгийн хувьд утсаны өөрийн Alarm апптай хослуулж ашиглахыг зөвлөж байна (вэб технологийн хязгаарлалт).

## 🔐 Эрхийн тогтолцоо ба аюулгүй байдлын анхааруулга
- Анх нээхэд **эхний админ** хэрэглэгчээ үүсгэнэ. Дараа нь админ баруун дээд булангийн дугуй (профайл) → *Хэрэглэгчид → + Нэмэх*-ээр шинэ хэрэглэгч нэмнэ.
- Хэрэглэгч бүрийн өгөгдөл (даалгавар, зуршил, сэрүүлэг) тусдаа хадгалагдаж, бусдад харагдахгүй.
- Админ хэрэглэгчийн төрөл, хандах эрхийг өөрчлөх, нууц үг шинэчлэх, устгах боломжтой. Сүүлчийн админыг устгах/бууруулахаас сэргийлсэн.
- ⚠️ **Анхаар:** Энэ нэвтрэлт нь браузер дотор (client-side) ажилладаг тул нэг төхөөрөмж дээрх хэрэглэгчдийг (жишээ: гэр бүлийн таблет) тусгаарлахад тохиромжтой. Браузерын хэрэгсэл (devtools) мэддэг хүний эсрэг "жинхэнэ" хамгаалалт **биш**. Үнэхээр найдвартай нэвтрэлт хэрэгтэй бол сервер талын баталгаажуулалт (backend) нэмэх шаардлагатай.

## Файлын бүтэц
| Файл | Зориулалт |
|------|-----------|
| `index.html` | Аппын бүтэц |
| `styles.css` | Загвар (харанхуй сэдэв) |
| `app.js` | Бүх логик (даалгавар, календар, зуршил, сэрүүлэг) |
| `manifest.json` | PWA тохиргоо (суулгах) |
| `sw.js` | Service worker (офлайн ажиллагаа) |
| `icons/` | Аппын дүрс зургууд |

## Мэдээлэл хадгалалт
Бүх өгөгдөл браузерийн `localStorage`-д хадгалагдана (`minii-tuslakh-v1` түлхүүр). Тухайн төхөөрөмж/браузерт л үлдэнэ. Браузерийн өгөгдөл цэвэрлэвэл устана.
